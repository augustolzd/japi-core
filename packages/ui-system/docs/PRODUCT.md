# Product: Japi Design System (ui-system)

## User Personas
* **Desarrolladores Frontend:** Equipos internos de Japifon que construyen Dashboards y Portales.
* **Product Designers:** Para asegurar la consistencia visual a través de toda la infraestructura de Japi.

## Business Value
* **Velocidad de Salida al Mercado (TTM):** Reduce el tiempo de desarrollo al reutilizar componentes pre-construidos y probados.
* **Consistencia de Marca:** Garantiza que la experiencia de usuario sea idéntica en el portal de clientes y en el dashboard administrativo.
* **Mantenibilidad:** Centraliza los cambios visuales en un solo lugar, impactando todas las aplicaciones simultáneamente.

## Feature List
* **Atomic Components:** Botones, Inputs, Modales y Tablas base con estilado premium.
* **Brand Variables:** Sistema de tokens de diseño para colores, tipografía y espaciado.
* **Zero-Runtime Styling:** Uso de CSS Modules para máximo rendimiento y cero peso extra de frameworks de utilidad.

## KPIs
* **Component Adoption Rate:** Porcentaje de interfaces que utilizan componentes del sistema vs estilos locales.
* **Bundle Performance:** El impacto del sistema de diseño en el tiempo de carga de las aplicaciones.
