# Product: Japi Marketing (Plataforma de Campañas)

## Misión de Negocio
Este portal es la interfaz visual para ejecutar toda la estrategia de comunicaciones masivas de la marca y clientes.

## User Personas
*   **Marketing Managers:** Crean campañas visuales, templates y dashboards.
*   **Community Managers:** Administran las listas de contactos y respuestas de audiencias.

## Feature List
1.  **Gestor de Campañas:**
    *   Interfaz Drag & Drop para creación de mensajes multicanal (Email, SMS, WA).
    *   Validación de Templates (HSM, RCS).
2.  **Segmentación:**
    *   Gestión de Bases de Datos (Audiencias).
    *   Filtros dinámicos (Ej: "Usuarios que abrieron el último email").
3.  **Analytics:**
    *   Dashboard en tiempo real: Open Rate, DLR, Clicks, Conversión.
    *   Heatmaps de Clicks (Email).
