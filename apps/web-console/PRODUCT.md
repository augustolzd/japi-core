# Japi IAM Console (El Cerebro)

**Perfil:** Desarrolladores y Administradores de IT.
**Carga:** Baja frecuencia, alta criticidad (escritura en DB).

## Misión
Centralizar la configuración lógica y de seguridad de todo el ecosistema Japi. Al estar separada, un error en las interfaces de usuario de otros portales no puede afectar accidentalmente la configuración de infraestructura crítica.

## Responsabilidades Clave
- **Control de Accesos:** Gestión de usuarios, asignación de roles jerárquicos y políticas IAM.
- **Configuración de Binds:** Gestión de parámetros técnicos para conexiones SMS, RCS y Email.
- **Seguridad Programática:** Rotación de API Keys y gestión de secretos para integraciones externas.
- **Aislamiento:** Garantizar que los cambios en la lógica de permisos sean atómicos y seguros.

## Ventaja Estratégica
La separación física garantiza que la "Capacidad de Mando" del sistema nunca se vea comprometida por el tráfico masivo de las campañas de marketing o las consultas pesadas de facturación.
