# Product: Japi Auth (web-auth)

## User Personas
* **Nuevos Usuarios:** Personas que desean registrarse para usar Japi Connect.
* **Usuarios Existentes:** Clientes y administradores que necesitan autenticarse.
* **Desarrolladores:** Necesitan una integración de SSO simple para sus aplicaciones.

## Business Value
* **Seguridad:** Centraliza la gestión de identidades y secretos.
* **Unified Experience:** Los usuarios solo inician sesión una vez para acceder a todo el ecosistema Japi (SSO).
* **Compliance:** Facilita la auditoría de accesos y la gestión de roles (RBAC).

## Feature List
* **User Registration:** Registro con validación de email.
* **Single Sign-On (SSO):** Autenticación centralizada distribuida a micro-sitios.
* **Role Based Access Control (RBAC):** Asignación de roles (User, Operator, Admin).
* **Self-Service Profile:** Gestión de contraseñas y datos básicos.

## KPIs
* **Auth Success Rate:** Éxito de logins vs fallidos (por error de sistema).
* **Onboarding Time:** Tiempo promedio desde registro hasta el primer acceso.
* **Active Sessions:** Número de sesiones concurrentes a través de la plataforma.
