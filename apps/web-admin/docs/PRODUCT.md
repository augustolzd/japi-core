# Product: Japi Gateway UI (Rutas & Config)

## Misión de Negocio
Este portal es el "Centro de Telecomunicaciones" donde los operadores definen cómo se enruta cada mensaje y cuánto cuesta.

## User Personas
*   **Ingenieros de Telecom (Operations):** Administran proveedores y rutas.
*   **Comerciales de Tráfico (Account Managers):** Establecen tarifas de costo y precio.

## Feature List
1.  **Configuración de Carriers:**
    *   Gestión de proveedores (Ej: Twilio, Syniverse, Integraciones P2P).
    *   Asignación de prioridades técnicas (Least Cost, Quality, Load Balance).
2.  **Gestor de Precios:**
    *   Definición de costos (Buy Rate) por País/MCC/MNC.
    *   Configuración de márgenes y precios de venta (Sell Rate).
3.  **Monitoreo de Rutas:**
    *   Visualización de tasas de entrega (SR) y latencia (P95, P99) en tiempo real por ruta.
