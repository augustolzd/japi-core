# Product: Japi Admin Dashboard

## User Personas
* **Administradores de Japi:** Personal interno responsable de configurar clientes, zonas y saldos.
* **Soporte Técnico:** Para revisar logs de auditoría y estados de tráfego.

## Business Value
* **Control Operativo:** Visibilidad total sobre la infraestructura y el consumo.
* **Autogestión:** Permite configurar reglas de negocio sin intervención de ingeniería.
* **Seguridad:** Panel centralizado con permisos granulares.

## Feature List
* **User Management:** Control de usuarios y permisos.
* **Telecom Config:** Gestión de zonas y reglas de rutas.
* **Billing Overview:** Visualización de balances y movimientos de centros de costos.

## KPIs
* **Operative Efficiency:** Tiempo promedio para dar de alta un cliente o zona.
* **Uptime:** Disponibilidad del panel de control.
